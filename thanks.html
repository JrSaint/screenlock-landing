First-launch checklist

  1. Mount the DMG by double-clicking it in Finder.
  2. Drag ScreenLock to Applications using the shortcut inside the DMG window.
  3. Open Applications → ScreenLock. First time you may see a Gatekeeper prompt asking if you want to open it — choose Open.
  4. Grant Accessibility permission in System Settings → Privacy & Security → Accessibility. ScreenLock will show you exactly where to click. This permission is required so it can lock keyboard, mouse, and trackpad input.
  5. Press your hotkey — the default is ⌃⌥⌘L — to lock and unlock. Customize it from the menu bar lock icon → Settings.
